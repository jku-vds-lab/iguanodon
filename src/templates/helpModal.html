Goal

Iguanodon now has Font Awesome icons

Iguanodon has three different games, the aim of each game is to create the perfect visualization in as few steps as possible.

Game

Each game is based on different literature on how to create the perfect visualization. They introduce guidelines on how and which design choices should be applied for a visualization. For each game you have fixed number of attempts to create the perfect visualization.

Game Board

The game board is split in two halves the upper half shows the visualization(s). In the lower half is interactive and allows you to change the visualization and get feedback on how good the created visualization is.

Visualization

There are two visualizations: the left one shows you the previous visualization. Only if you have already attempted to solve the game once. The right visualization is a preview of your current attempt and can be changed by toggling actions.

Action

An action is a simple visualization design choice, which is either applied or not applied to the visualization.

Each action belongs to one objective, but each objective can have multiple actions.

Objective

Each game has multiple objectives, each objective represents a part of the goal to create a perfect visualization.

To fulfill an objective all of its relevant actions have to be applied in the right way. A game is won when all objectives have been fulfilled.

Gameplay

  1. You toggle as may actions as you like to create the a visualization, that you think is a perfect solution.
  2. Click the -Confirm- button to check your visualization. The state of the objectives will give you feedback about the visualization.
  3. If not all objectives were fulfilled (green) you can try again to create the perfect visualization in the next attempt by modifying the actions.

You win the game if all actions are applied correctly, or rather if all objectives are fulfilled. For each game a high score will be shown next to the different games, if you solved the game fast you will get different rewards: gold, silver, or bronze badge.

If you could not solve the game within the given number of attempts the game is over and you be provided with one solution.

Note: For some games multiple solutions exist, if you really want to test your skill you could try to find all possible solutions.

Dataset

The dataset is about different cars and their properties and was created by Vega. It consists of 406 cars with 9 columns.
