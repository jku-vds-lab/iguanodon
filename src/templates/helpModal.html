Goal

The application has three different game variations and the aim of each game variation is to create a good visualization in as few steps as possible.

Game Variations

Each game variation is based on different literature on how to create good visualizations. They introduce guidelines for design choices on how they should be applied to visualizations. For each game variation, you have a fixed number of attempts to create a good visualization.

Game Board

The game board is split into two halves, the upper half shows the visualization(s). The lower half is interactive and allows you to change the visualization and get feedback on how good the created visualization is.

Visualization

There are two visualizations: the left one shows you the previous visualization if you have already attempted it once. The right visualization is a preview of your current attempt and can be changed by toggling actions.

Action

An action is a simple visualization design choice, which is either applied or not applied to the visualization.

Each action belongs to one objective, but each objective can have multiple actions.

Objective

Each game variation has multiple objectives, and each objective represents a part of the goal to create a good visualization.

To fulfill an objective all of its relevant actions have to be applied in the right way. A game variation is won when all objectives have been fulfilled.

Gameplay

  1. You toggle as many actions as you like to create a visualization, that you think is a good solution.
  2. Click the "Confirm" button to check your visualization. The state of the objectives will give you feedback about the visualization.
  3. If not all objectives were fulfilled you can try again to create a good visualization in the next attempt by modifying the actions.

You win the game variation if you fulfill all the objectives. For each game variation, a high score will be shown next to the different games. If you solve the game variation in one, two, or three attemps, you get a gold, silver, or bronze badge, respectively.

If you could not solve the game variation within the given number of attempts the game is over and you be provided with one solution.

Note: For some game variations, multiple solutions exist. If you want to test your skill, you could try to find all possible solutions.

Dataset

The dataset is about different cars and their properties and was created by the American Statistical Association (ASA) and is also used by Vega. It consists of 406 cars with 9 columns.
